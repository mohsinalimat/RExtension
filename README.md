# RExtension
Frequently used extension classes which include some convenience methods and manipulations.

# How to use
Just add Extensions folder into your project.

# Extensions
* UIView
* UIImage
* UIImageView
* UITableView
* UITextField
* UIViewController
* UIWindow
* UIButton
* UINavigationController
* String
* UISlider
* Dictionary
* Array
